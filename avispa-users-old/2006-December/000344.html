Dear all AVISPA users,


we are very happy to announce the first release of

        SPAN -- a Security Protocol ANimator for AVISPA


SPAN is designed to help protocol developers in writing HLPSL
specifications. From an HLPSL specification SPAN helps in
interactively buiding  Message Sequence Charts (MSC) of the protocol
execution. Since SPAN implements an active intruder, it can
also be used to interactively find and build attacks over protocols.

SPAN includes a local graphical interface similar to the one of
AVISPA.

-------------------------------------------------------------------
SPAN is distributed under LGPL in source form. We also provide
'complete binary packages' that run on **Linux, Windows, MacOS**
with SPAN and binary versions of AVISPA tools (when available
on a given architecture).

Source and binary packages can be downloaded here:

http://www.irisa.fr/lande/genet/span


Please send bugs, comments, improvements to genet at irisa.fr
