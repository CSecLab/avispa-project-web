The AVISPA Library of protocols
This document presents the specifications of the protocols and security problems modelled in the HLPSL by the AVISPA Team and analysed with the AVISPA Tool, called the AVISPA Library. This set of protocols is a large subset of those described in Deliverable 6.1: List of selected problems.
For each of the protocols, we describe its purpose, the message exchange in an Alice&Bob-style notation, the corresponding security problems and their classiciation, any attacks found, finally the actual HLPSL code. Where appropriate, we give further explanations and comments.
External contributions to the library are welcome; please send them to the avispa-users mailing list.
