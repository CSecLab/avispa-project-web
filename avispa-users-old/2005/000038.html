Anil,

It would appear that you are using the web interface 
(http://www.avispa-project.org/web-interface/) it is possible to 
download and run AVISPA locally. I would recommend this if you plan to 
use the tool seriously.

As for the messages you are receiving
OFMC    : SAFE
CL-AtSe : SAFE
SATMC   : SAFE
TA4SP   : SAFE

OFMC, CL-AtSe, SATMC and TA4SP simply refer to the backend in use. The 
online tool will attempt to execute your protocol with each backend in turn.

If you download the tool it will come complete with documentation, which 
provides a good starting point for understaning the tool and will 
include further explanation.
